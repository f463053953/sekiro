# 前言

vue 用了很久，一直没有深入了解这些基本的东西，最近闲下来了准备系统的学习这些基础知识，于是我开始学习 VUE 的生命周期

# vue 生命周期简介

VUE2.0Description
beforeCreate组件实例刚被创建，组件属性计算之前如 data 属性等
created组件创建完成，属性绑定完成，但 dom 元素还没生成，$el 属性还不存在
beforeMount模板编译 / 挂载之前
mounted模板编译 / 挂载之后
beforeUpdate组件更新之前
Update组件更新之后
activated组件被激活时候调用
deactivated组件被移除时候调用
beforeDestroy组件销毁前调用
destroyed组件销毁后调用

# 举个栗子

基于 vue2.0 来说，生命周期分为 beforeCreate,created,beforeMount,mounted,beforeUpdate,Update,beforeDestroy,destroyed
结合以下代码可以直观看出各个阶段触发

beforecreated：el 和 data 并未初始化
created: 完成了 data 数据的初始化，el 没有
beforeMount：完成了 el 和 data 初始化
mounted ：完成挂载
